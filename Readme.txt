Mind Mapping software
---------------------

Purpose: To serve as a note taking aid for students and professionals. 
The target user will use this for better organizing notes in a hierarchical 
manner and better planning.

Features -list 05/10/2015
-------------------------
Create mind maps
Edit mind maps
	- add node
	- delete node
	- add child nodes
	- drag and drop nodes
	- edit nodes
	- Styling of nodes
File management
	- Save mind map
	- Delete mind map
	- View/Edit older mind map
Added features
	- Download
	- share
	- Animated replay of mindmap
